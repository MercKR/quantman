## 주요 변경

### 미국 실시간 시세 — D-prefix 분기 지원
- KIS WebSocket 해외 시세 prefix를 사용자 설정으로 분기:
  - **OFF (default)**: BAQ/BAY/BAA (15분 지연시세, 신청 불필요)
  - **ON**: DNAS/DNYS/DAMS (진짜 실시간, KIS HTS [7781] 무료 신청 필요)
- 로컬앱 GUI에 "🇺🇸 미국 실시간 시세 사용" 체크박스 + "신청 방법" 버튼 추가.
- 알림 분기:
  - OFF + tick 없음 → "지연시세 사용 중, 무료 신청 안내" 메시지
  - ON + tick 없음 → "신청 처리 대기 또는 거부 상태 확인" 메시지

### 사용자 액션 — 미국 실시간 시세 무료 신청 (선택)
미국 종목 장중 실시간 손절을 사용하려면:

1. **KIS HTS [7781]** 또는 **모바일앱 → 고객지원 → 거래서비스 신청 → 해외증권 → 해외 실시간시세 신청**에서 미국 무료 시세 신청.
   - 비용: 일회성 ~1,500원 처리비. 월 정기 결제 없음.
   - 처리: 약 1영업일.
2. 로컬앱 GUI에서 "🇺🇸 미국 실시간 시세 사용" 체크박스 켜기.
3. 다음 미국 세션부터 진짜 실시간 시세로 익절/손절/트레일링 동작.

신청 안 한 경우: 미국 종목은 장 마감 후 settlement cycle에서만 청산 평가됩니다 (한국 종목은 영향 없음 — 기본 실시간).

이 release는 모든 server 버전과 호환. 변경은 로컬앱 + KIS WebSocket prefix만.

## 알려진 한계
- KIS API 약관상 시세 정보는 본인 투자 목적 전용. 제3자 재배포 금지 (서버가 받아 push하는 SaaS 모델은 KOSCOM 별도 라이선스 필요).
- 지연시세 → 실시간 전환은 다음 미국 세션 cycle 시점부터 적용. 진행 중 US loop는 재시작 후 적용.
